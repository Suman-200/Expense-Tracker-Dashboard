## 📌 Project Overview

A feature-rich expense tracking dashboard with budget management, category-wise analysis via a donut chart, and transaction history with filtering — all in a dark-themed UI.

## ✨ Features

- **Set Monthly Budget** — Input your budget and track usage
- **Add Expenses** — With description, amount, category, and date
- **Budget Progress Bar** — Visual indicator with color-coded alerts
- **Donut Chart** — Category-wise spending breakdown (Canvas API)
- **Transaction Table** — Filterable by category, with delete option
- **Summary Cards** — Total budget, spent, remaining, count
- **Persistent Storage** — localStorage saves all data
